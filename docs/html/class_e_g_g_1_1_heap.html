Detailed Description

Base Heap class.

EGG::Heap is not concrete and only acts as an interface for actual implementations.

◆ Heap()

EGG::Heap::Heap ( MEMiHeapHead *  heapHandle)

Heap constructor.

Creates a Heap wrapper around the supplied MEM heap.

  • Calls down to the parent Disposer constructor before setting values.
  • Parent members _14, _18, _1C are zeroed; mName is set to "NoName".
  • ut::List at _28 initialized.
  • This heap is added to the global heap list (sHeapList).
Parameters
[in]heapHandleMEM heap handle to create EGG::Heap wrapper around.
Precondition
initialize() has been called. (WS asserts sHeapInitialized)
Returns

◆ findContainHeap()

Heap * EGG::Heap::findContainHeap ( const void *  memBlock)
static

Scan global heap list for heap containing specified memory block.

First runs MEMFindContainHeap() to determine if it is in a MEM heap. If this check passes, it will iterate through sHeapList, checking if the heap in each iteration's _10 matches the MEM heap found before.

Parameters
[in]memBlockMemory block to look scan for container.
Precondition
sHeapList is intialized. (sIsHeapListInitialized)
Returns
The Heap that contains the memory block.
