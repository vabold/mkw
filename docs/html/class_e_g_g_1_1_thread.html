)
static

The callback for switching threads.

Finds the associated EGG threads of the two OSThreads, and calls onExit() on the first thread and onEnter() on the next.

Parameters
[in]fromThe last thread running.
[in]toThe next thread to run.
Note
Either threads may be null if there is no thread to run. This function handles this accordingly
