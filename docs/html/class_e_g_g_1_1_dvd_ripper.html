enum EGG::DvdRipper::EAllocDirection

Describes the direction of allocating new blocks in a free memory region.

Enumerator
ALLOC_DIR_TOP 

[1] Negative alignment; allocate block from top of free block.

ALLOC_DIR_BOTTOM 

[2] Positive alignment; allocate block from bottom of free block.

Load a file on the disc to main RAM given the path.

Parameters
[in]pathPath to the file on the disc.
[in,out]dstDestination buffer to write file. If nullptr, an appropriately sized block of memory will be allocated. Otherwise the pre-allocated block MUST be big enough to store the file.
[in]heapHeap to use if allocating a block. Cannot be nullptr if dst is also nullptr.
[in]allocDirectionAllocation direction of dst if it's nullptr.
[in]offsetOffset of the file on the disc to start reading from.
[out]amountReadIf not nullptr, will be set to the amount of bytes read from the disc.
[out]fileSizeIf not nullptr, will be set to the file size of the file on disc (unrounded/offset).
Returns
Pointer to the ripped file, or nullptr in case of failure.

Load and decompress a file on the disc to main RAM given the path.

Parameters
[in]pathPath to the file on the disc.
[in]streamDecompUninitialized decompressor.
[in,out]dstDestination buffer to write file. If nullptr, an appropriately sized block of memory will be allocated. Otherwise the pre-allocated block MUST be big enough to store the DECOMPRESSED file.
[in]heapHeap to use if allocating a block. Cannot be nullptr if dst is also nullptr.
[in]allocDirectionAllocation direction of dst if it's nullptr.
[in]offsetOffset of the file on the disc to start reading from.
[in]sizeIf non zero, size of the file to read. The sum of offset and size MUST NOT be larger than the compressed file size.
[in]maxChunkSizeThe maximum size of the temporary buffer that will be allocated to read the compressed file.
Returns
Pointer to the ripped and decompressed file, or nullptr in case of failure.
